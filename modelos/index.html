Para incluir novos modelos, você precisa adaptar a função executarMonitoramento no arquivo script.js para ler os dados a partir do código do NetLogo.

Para identificar corretamente como pegar os dados do modelo, é necessário abrir o código do modelo na aba Netlogo Code e analisar o início do modelo. Por exemplo, no Lobo e Ovelhas, o código tem a detalhação do breed informando o nome dos membros. Já no Traffic Basic, os carros estão armazenados na variável padrão turtle já que não foi declarado como breed. Essa análise permite identificar os nomes das variáveis que devem ser adaptadas na função executarMonitoramento para cada novo modelo.
